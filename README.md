## 🗂 Tool Catalog

Welcome to the `P.MIROSHNIK.TOOLS` repository. This is a central hub for various scripts and utilities designed to accelerate and automate routine tasks in graphic design and prepress production (Adobe Photoshop, Illustrator, etc.).

| Tool | Description | Status |
|:---|:---|:---:|
| **[AddSpotWhite](https://github.com/v1per4ever/AddSpotWhite)** | Automatic generation of Spot White channels for print | Active |
| **[Print-Prep-Orchestrator](https://github.com/v1per4ever/Print-Prep-Orchestrator)** | Batch export and preflight pipeline for Illustrator | Active |
| **[ps-bulk-mockups-filler](https://github.com/v1per4ever/ps-bulk-mockups-filler)** | Mass generation and filling of mockups in Photoshop | Active |
| **[barcode-pdf-generator](https://github.com/v1per4ever/barcode-pdf-generator)** | Generate barcodes and compile them into PDFs (Python) | Active |

> 🚀 **New tools are added regularly!** Stay tuned for updates on my profile.

## 💻 Installation
Each script or utility has its own specific requirements. For most Adobe scripts (.jsx), simply place them in the application's scripts folder:
- **Windows:** `C:\Program Files\Adobe\Adobe [App Name] [Version]\Presets\Scripts\`
- **macOS:** `/Applications/Adobe [App Name] [Version]/Presets/Scripts/`

For detailed instructions, refer to the README of each respective repository.
